# 📌 Общая информация

Вам требуется выполнить анализ данных с помощью библиотеки **NumPy**. В ходе работы необходимо создавать и обрабатывать массивы, применять поэлементные и матричные операции, склеивать и разбивать массивы, использовать статистические функции и булевы маски.

**Все исходные данные должны быть случайными.** Для проверки расчетов можно использовать приведённые примеры, но для закрепления навыков следует генерировать свои массивы случайным образом.

Задача включает следующие направления работы: 
* создание вспомогательных массивов 
* анализ структуры массивов 
* склеивание массивов 
* работа с единичными данными 
* оценка динамики изменений
* вычисление статистических показателей
